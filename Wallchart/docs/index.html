AHC Wallchart Control (HTML5 Canvas)

library-wallchart

Description

This is a control library for a wallchart control used for scheduling.

Usage

See demo application in source (src) directory.

Depends On

All javascript dependancies.

External

Code

  1. Fabric.js

Tests

  1. Please see package.json

Internal

You will need to import 3 scripts for the use of this control.

  1. ahc.{version}.js - A utilities library for all AHC frontends.
  2. ahc.model.{version}.js - A simple scheduling domain model.
  3. ahc.wallchart.{version}.js - The wallchart control control library.

Tests

Please feel free to run the jasmine test located in the src directory.
